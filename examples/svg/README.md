# Example SVGs

These stroke-only SVGs are tested with the LightElf Laser SVG converter and are
intended as starter files for **Configure > Upload SVG**.

- `star.svg`: simple cyan outline.
- `home_assistant.svg`: two-color Home Assistant-inspired outline.
- `lightelf_home.svg`: compact LightElf beam/house mark.
- `svideo_logo.svg`: detailed two-color round logo; heavier than the others.

For best results, keep SVGs stroke-only and use simple path geometry. Supported
stroke colors include red, green, blue, yellow, cyan, purple/magenta, and white.
