# Third-Party Notices

## Hershey Font Data

This repository bundles selected Hershey vector font data files under:

```text
custom_components/lightelf_laser/fonts/*.jhf
```

Source used for this project:

- https://github.com/kamalmostafa/hershey-fonts
- Revision: `1356bf2f83d380fcef68c887e88675eb9d445d86`

The `hershey-fonts` library/tools are GPLv2+, but this project bundles only the
`hershey-fonts/*.jhf` glyph data. The upstream README describes the glyph data
as covered by a permissive use and redistribution license.

The original Hershey distribution note says the font data may be used by anyone
for any purpose, commercial or otherwise, provided the acknowledgements below
are distributed with the data.

Required acknowledgements:

- The Hershey Fonts were originally created by Dr. A. V. Hershey while working at
  the U.S. National Bureau of Standards.
- The format of the font data in this distribution was originally created by
  James Hurt of Cognition, Inc.
